== Description ==

Альфа-Банк "Купи Легко" - плагин выводит кнопку "Купи Легко" от Альфа-Банк на странице корзины WooCommerce.

[Описание и подключение услуги на сайте Альфа-Банк](https://anketa.alfabank.ru/kupilegko/)

== Frequently Asked Questions ==

= Какие существуют ограничения? =

1. Цена одного товара - не менее 10 рублей.
2. Количество товаров - не более 30 шт.
Дополнительные ограничения по работе с системой вы можете найти в инструкции для подключения партнеров к системе «Купи Легко».

== Installation ==

Установите плагин через панель администратора WordPress.

1. Для работы плагина **Alfa Bank Buy Easy** необходимо наличие установленного и активированного плагина **WooCommerce**.
2. Настройки плагина находятся в меню: `WooCommerce -> Настройки -> Alfa Bank Buy Easy`.
